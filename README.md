# Notification for entered and left on Discord

# Version
## Python
3.6

## discord.py
1.2.0a

# Usage

## Build
```bash
$ docker-compose up -d --build
```

## Run container
```bash
$ docker-compose up -d
```

## Stop container
```bash
$ docker-compose down
```

## Run Hello World!
```bash
$ docker-compose run --rm bot python hello_world.py
```

## Run Entering / Learing manage bot
```bash
$ docker-compose run --rm bot python bot.py
```
